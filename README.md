# Property Price Prediction App

A simple, beginner-friendly Streamlit web application that predicts property prices based on various features like bedrooms, bathrooms, and property type.

## Dataset

Dataset: https://www.kaggle.com/datasets/jakewright/house-price-data

Download and place the dataset (csv file) in the data folder.

## Install

```bash
pip install -r requirements.txt
or
pip3 install -r requirements.txt
```

## Train model

```bash
python train_model.py
or
python3 train_model.py
```

## Run app

```bash
streamlit run app.py
or
python3 -m streamlit run app.py
```
